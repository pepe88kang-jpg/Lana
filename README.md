# Lana

Undangan pernikahan digital untuk Ega Lana Wibowo dan Karunia Riski Apri Hesi.

## Tentang

Website ini dibuat dengan Next.js dan siap untuk dideploy ke Vercel. Halaman utama menampilkan informasi mempelai pria dan wanita, serta panduan souvenir dan kontak WhatsApp admin.

## Deployment

1. Pastikan repository sudah ada di GitHub.
2. Buat project Vercel dan hubungkan ke repository ini.
3. Tambahkan environment variables di Vercel:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Deploy dan buka URL Vercel yang diberikan.

## Supabase

Buat table `souvenir_requests` di Supabase jika ingin menyimpan data permintaan souvenir.
